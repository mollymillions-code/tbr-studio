## How to Use This File

- **Add entries** as events happen (race results, team changes, milestones, external events).
- **Tag every entry** with one or more of: `#race`, `#team`, `#pilot`, `#tech`, `#conservation`, `#culture`, `#milestone`.
- **Include context** that would help a content creator understand the moment.
- **Link to knowledge files** when relevant (e.g., `→ See /knowledge/arcs/characters/masha.md`).
- **Never delete entries.** This is an append-only log. The agent needs the full history.
