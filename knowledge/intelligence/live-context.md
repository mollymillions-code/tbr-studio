# Live Context — Real-Time Intelligence

This file is the agent's awareness of the world OUTSIDE of TBR.
It should be updated before every content creation session.

## How the Agent Uses This File

Before creating any content, the agent should:
1. **Web search** for current trends, pop culture, viral moments
2. **Check** what is trending on social platforms (Instagram, Twitter, TikTok)
3. **Identify** cultural moments that TBR content could connect to
4. **Update this file** with findings
5. **Reference relevant context** when creating content

The goal: TBR content should feel CURRENT. Not created in a vacuum.
Topicality makes content feel alive.
